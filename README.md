# Mini-Projet : Système de Gestion d’Étudiants avec Analyse de Performances

Ce projet est un mini-système en **Python 3.10+** permettant de gérer des étudiants, leurs notes et de générer des statistiques.

## Fonctionnalités
- Ajouter, modifier, supprimer un étudiant
- Calculer les moyennes individuelles et par matière
- Afficher les étudiants avec une moyenne > 15
- Générer un rapport global (JSON-like)
- Gérer les erreurs avec `try/except`

## Technologies utilisées
- Python 3.10+
- Structures de données : `list`, `dict`, `set`, `tuple`, `frozenset`

## Installation
Cloner le projet :
```bash
git clone https://github.com/Fadhel642/mini-projet-gestion-etudiants.git
cd mini-projet-gestion-etudiants*
```
---

## Exécution

```
python src/gestion_etudiants.py
```
